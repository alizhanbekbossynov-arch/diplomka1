# Менеджер задач (Task Manager)

## Описание проекта

Менеджер задач— консольное приложение, разработанное на языке C++. Программа позволяет пользователю создавать и удалять задачи, сохранять их между запусками программы, а также вести список дней рождения.

Для каждого пользователя создаётся отдельный аккаунт, благодаря чему его задачи хранятся отдельно от задач других пользователей.

---

## Функционал

Программа предоставляет следующие возможности:

* Регистрация нового пользователя.
* Авторизация существующего пользователя.
* Добавление задач.
* Просмотр списка задач.
* Удаление задач.
* Автоматическое сохранение и загрузка задач.
* Добавление дней рождения.
* Просмотр списка дней рождения.
* Удаление дней рождения.
* Автоматическое сохранение и загрузка дней рождения.
* Проверка дней рождения при запуске программы.
* Проверка корректности вводимых данных.
* Хранение данных каждого пользователя в отдельных текстовых файлах.

## Инструкция по запуску

1. Скачать или клонировать репозиторий.(git clone https://github.com/alizhanbekbossynov-arch/diplomka1.git)
2. Открыть проект в Microsoft Visual Studio.
3. Собрать проект (Build → Build Solution).
4. Запустить программу (Debug → Start Without Debugging или клавиша Ctrl + F5)
